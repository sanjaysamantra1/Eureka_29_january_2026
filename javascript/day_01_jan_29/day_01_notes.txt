Web Development
================
-Developing Website

1. FrontEnd : User Interface  ( Look & Feel )
        HTML : Content ( text , paragraph , image , table , audio , video )
        CSS : Styling ( border , backgroundcolor , width, height, margin )
        Bootstrap : CSS library ( twitter ) ( pre-defined css classes )
        Javascript : Behaviour 
        Angular/React : Javascript Frameworks ( RAD , CLI, Routing , Form Handling )
2. BackEnd: Develop REST APIs ( Java / .net / python / PHP / Go / Nodejs)
3. DataBase : Holds the data ( oracle , mysql, mongoDB )
