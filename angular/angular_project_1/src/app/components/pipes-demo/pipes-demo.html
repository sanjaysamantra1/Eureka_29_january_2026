Let's Learn Angular PIpes

uppercase, lowercase, titlecase:
userName : {{userName}}
userName : {{userName | uppercase}}
userName : {{userName | lowercase}}
userName : {{userName | titlecase}}
currency pipe:
{{ salary }}
{{ salary | currency }}
{{ salary | currency : 'USD' }}
{{ salary | currency : 'INR' }}
{{ salary | currency : '€' }}
{{ salary | currency : '¥' }}
{{ salary | currency : 'USD' : 'code' }}
{{ salary | currency : 'USD' : 'symbol' }}
Date pipe:
{{ dateObj | date }}
{{ dateObj | date :'short' | uppercase}}
{{ dateObj | date :'medium'}}
{{ dateObj | date :'long'}}
{{ dateObj | date :'full'}}
==================
{{ dateObj | date : 'shortDate' }}
{{ dateObj | date : 'mediumDate' }}
{{ dateObj | date : 'longDate' }}
{{ dateObj | date : 'fullDate' }}
=================
{{ dateObj | date : 'shortTime' }}
{{ dateObj | date : 'mediumTime' }}
{{ dateObj | date : 'longTime' }}
{{ dateObj | date : 'fullTime' }}
==============
{{ dateObj | date : 'dd-MMM-yyyy' }}
{{ dateObj | date : 'hh:mm:ss a' }}
{{ dateObj | date: "fullTime":"UTC" }}
{{ dateObj | date: "fullTime":"IST" }}
Number & percent pipe
{{ 12.111222333 | number: "3.2-5" }}
{{ 12.1 | number: "3.2-5" }}
{{0.5 | number:'3.2-5'}}
{{0.5 | percent}}
{{0.75 | percent}}
json pipe
{{user}}
{{user | json}}
key-value pipe
@for(entry of user | keyvalue ;track $index){
{{entry.key}}---{{entry.value}}
}
-slice pipe can be used with array and string
{{cars | slice : 2 : 4}}
{{'Angular is a Framework' | slice : 0 : 7}}
{{'Angular is a Framework' | slice : 0 : -4}}
Custom Pipe(remaining):
{{msg}}
Remaining: {{msg | remaining}}
Custom Pipe(ordinal):
Input: {{num}}
Ordinal: {{num | ordinal}}

@for (item of arr | mysort; track $index) { {{item}}   }
